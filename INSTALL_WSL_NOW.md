# Install WSL - Quick Guide

## Step 1: Open PowerShell as Administrator

1. Press `Windows Key`
2. Type "PowerShell"
3. **Right-click** on "Windows PowerShell"
4. Click "**Run as administrator**"

## Step 2: Run Installation Command

In the Administrator PowerShell, paste this:

```powershell
wsl --install
```

This will:

- Install WSL 2
- Install Ubuntu Linux
- Take about 5-10 minutes

## Step 3: Restart Computer

After installation completes, restart your computer.

## Step 4: Launch Ubuntu

After restart:

1. Open Start Menu
2. Search for "Ubuntu"
3. Click to open
4. Wait for setup (first time only)
5. Create a username and password when prompted

## Step 5: Verify It Works

In Ubuntu terminal, run:

```bash
echo "WSL is working!"
```
